# Time period definitions

## Names for different time periods

Different time periods use the names shown in the table below. All clock times are local time.

| Finnish (phrase key) | English meaning |
| --- | --- |
| Aamu | Morning |
| Aamupäivä | Before noon |
| Keskipäivä | Midday |
| Iltapäivä | Afternoon |
| Ilta | Evening |
| Iltayö | Early night |
| Keskiyö | Midnight |
| Aamuyö | Late night / small hours |
| Päivä | Day |
| Yö | Night |

The [timeperiods.php](timeperiods.php) script renders the layered time-of-day chart used in the original documentation. Hour ranges (local time):

| Period | Hour range |
| --- | --- |
| Aamu (Morning) | 06–09 |
| Aamupäivä (Before noon) | 09–11 |
| Keskipäivä (Midday) | 11–13 |
| Iltapäivä (Afternoon) | 13–18 |
| Ilta (Evening) | 18–22 |
| Iltayö (Early night) | 22–00 |
| Keskiyö (Midnight) | 00–03 |
| Aamuyö (Late night) | 03–06 |
| Päivä (Day, coarse) | 06–18 |
| Yö (Night, coarse) | 18–06 |
