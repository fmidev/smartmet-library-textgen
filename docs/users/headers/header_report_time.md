# Header type "report_time"

Produces a header of the form "Sääennuste tiistaina kello 18". The clock time is taken from the start of the period. The type pairs particularly well with the period type "now".

Possible headers:

1. "Sääennuste N-na kello K".

In Swedish:

1. "Väderraport N-na klockan K."

In English:

1. "Weather report N-na K o'clock."
