# Story "none"

> **Status:** Specialized. Emits nothing.
>
> **Owner:** `SpecialStory::none()`.
> **Implementation:** `textgen/special_none.cpp` (~43 LOC).

Returns an empty `Paragraph`. Useful as a placeholder in content lists —
for example, you want a section to exist (so its header is emitted) but
have no body text yet.

## Configuration

None.
